# CLAUDE.md

## 进度汇报指令

执行任何包含多个 Task 的实施计划时，必须遵守以下规则：

1. 任务开始前，先输出完整任务列表（全部标记为 `⬜ 未进行`）
2. 开始一个 Task 时，将该 Task 标记为 `🔵 进行中` 并输出进度面板
3. 完成一个 Task 后，立即将该 Task 标记为 `✅ 已完成`，下一个 Task 标记为 `🔵 进行中`，输出进度面板
4. 进度面板格式固定如下：

```
## 执行进度

| Task | 内容 | 状态 |
|------|------|------|
| Task 1 | xxx | ✅ 已完成 |
| Task 2 | xxx | 🔵 进行中 |
| Task 3 | xxx | ⬜ 未进行 |

📌 当前：Task 2 | ▶️ 下一个：Task 3
```

5. 状态符号：`✅` 已完成 `🔵` 进行中 `⬜` 未进行 `❌` 失败
6. 不要只在全部任务结束后才汇总，每完成一个 Task 就必须输出一次
7. 如果 Task 失败，标记为 `❌`，并在面板下方说明失败原因和下一步建议
