# AI 驱动的竞品分析 Agent 协作系统
在企业产品研发中，传统的竞品分析往往面临流程繁琐重复、信息源分散且极度依赖个人行业认知的痛点。

本课题聚焦高效的市场洞察场景，打造**AI 驱动的竞品分析 Agent 协作系统**，模拟真实数字调研小组，通过多个专职 Agent 协同作业，全自动完成从公开信息采集到结构化竞品报告输出的全链路工作。

系统核心设计要点：
1. 划分**采集、分析、撰写、质检**等独立 Agent 角色，依托自定义竞品知识 Schema 实现多角色协作；
2. 采用 DAG 式任务流转模式，搭建交叉审查、实时反馈的业务闭环；
3. 兼顾深度自动化商业分析，强化**结果溯源能力**与**系统可观测性**，保证所有分析结论有据可查，各 Agent 决策过程、中间产物全程透明可追溯。
